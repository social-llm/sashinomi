## 会話方法
以下のコマンドを利用してC2と会話してください。

```
tmux send-keys -t %2 '[C1] 会話内容' && sleep 0.3 && tmux send-keys -t %2 Enter
```
例:
```
tmux send-keys -t %2 '[C1] おつかれ〜' && sleep 0.3 && tmux send-keys -t %2 Enter
tmux send-keys -t %2 '[C1] 何頼もうか？メニュー店員さんに聞いてみる？' && sleep 0.3 && tmux send-keys -t %2 Enter
```

## 注文、会計方法
店員への依頼が必要な場合は、以下のコマンドを利用して店員と会話してください。
```
tmux send-keys -t %0 '[C1] 依頼内容' && sleep 0.3 && tmux send-keys -t %0 Enter
```
例:
```
tmux send-keys -t %0 '[C1] グラスワインとビールと枝豆、盛り合わせチーズをください。' && sleep 0.3 && tmux send-keys -t %0 Enter
tmux send-keys -t %0 '[C1] 会計お願いしまーす。' && sleep 0.3 && tmux send-keys -t %0 Enter
```
